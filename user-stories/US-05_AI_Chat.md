# US-05 — Neura AI Companion Chat

---

## US-05-01 · Send a Text Message to Neura
**As a** Student,
**I want to** type a question and get a response from Neura in Egyptian Arabic,
**so that** I can get study help without needing a private tutor.

### Acceptance Criteria
- [ ] Chat input is RTL with Arabic placeholder: "اسأل نيورا..."
- [ ] Messages sent to backend → GPT-4o-mini with Socratic system prompt
- [ ] Neura's response is in Egyptian Arabic (العامية الراقية)
- [ ] Loading indicator shown while waiting for response
- [ ] Error state handled gracefully: "معلش، في مشكلة. حاول تاني بعد شوية"
- [ ] Chat history persisted locally (WatermelonDB) and synced to server

**Priority:** P1

---

## US-05-02 · Socratic Guardrail (Anti-Cheating)
**As a** Student,
**I want to** ask Neura to solve my homework,
**so that** I can get help — but Neura should guide me, not do it for me.

### Acceptance Criteria
- [ ] Neura NEVER provides a direct answer to a homework/exam question
- [ ] Neura provides the first hint or step and asks: "إيه اللي تفتكر يجي بعد كده؟"
- [ ] For math/physics: Neura states the relevant rule/formula and asks the student to apply it
- [ ] Guardrail is enforced at the backend system prompt level (not just frontend)
- [ ] If student explicitly asks "حلها عني" → Neura responds: "مش هينفع أحلها عنك، بس أنا هنا أساعدك توصل للحل بنفسك 💪"

**Priority:** P1

## US-05-03 · Voice Message to Neura (Feynman Technique)
**As a** Student,
**I want to** explain a concept out loud to Neura via voice,
**so that** I can practice the Feynman Technique and get feedback on what I missed.

### Acceptance Criteria
- [ ] Microphone button in chat input bar
- [ ] Hold to record; release to send
- [ ] Audio sent to Whisper API for Arabic transcription
- [ ] Transcription displayed as the student's message in the chat
- [ ] Neura analyzes the transcription against the topic context and responds with: missing keywords, misconceptions, and encouragement
- [ ] Works only when online; offline → show "محتاج إنترنت للميزة دي"

**Priority:** P2

---

## US-05-04 · Neura Voice Notes (TTS)
**As a** Student,
**I want to** hear Neura's responses as voice notes,
**so that** I can listen while taking notes or resting my eyes.

### Acceptance Criteria
- [ ] Each Neura message has a "🔊 استمع" button
- [ ] Tapping it plays TTS audio generated by ElevenLabs or OpenAI TTS in Arabic
- [ ] Audio is cached locally after first play to avoid re-fetching
- [ ] Playback can be paused/stopped
- [ ] Available for Pro users only; Free users see a "Pro ⚡" lock icon

**Priority:** P3

---

## US-05-05 · Chat History
**As a** Student,
**I want to** scroll back through my previous conversations with Neura,
**so that** I can review explanations I received earlier.

### Acceptance Criteria
- [ ] Chat history loads from local DB on screen open
- [ ] Paginated — loads 20 messages at a time, older messages load on scroll up
- [ ] History is private — not visible to parents or tutors
- [ ] Student can clear chat history from Settings

**Priority:** P2

---

## US-05-06 · Subject Context Tagging
**As a** Student,
**I want to** tag my chat session with a subject (e.g., Math, Chemistry),
**so that** Neura gives more relevant, subject-specific guidance.

### Acceptance Criteria
- [ ] Subject selector shown at the top of the chat screen
- [ ] Selected subject is passed as context to the AI prompt
- [ ] Subject tag is saved with the chat session for analytics
- [ ] Default is "عام" (General) if no subject is selected

**Priority:** P2
