# US-13 — Monetization & Subscription

---

## US-13-01 · Free Plan Limits
**As a** Student on the Free plan,
**I want to** understand what I can and can't do for free,
**so that** I can decide if upgrading is worth it.

### Acceptance Criteria
- [ ] Free plan includes: Pomodoro timer, To-Do list, Gamification, max 50 pages PDF/month, basic AI chat (limited messages/day)
- [ ] When a limit is hit → friendly upsell message: "وصلت للحد المجاني، ترقى لـ Pro وفك القيود ⚡"
- [ ] Upsell is non-intrusive — shown once per session, not on every action
- [ ] Free features never removed without notice

**Priority:** P1

---

## US-13-02 · Upgrade to Neura Pro
**As a** Student,
**I want to** upgrade to Neura Pro,
**so that** I get unlimited AI features and detailed analytics.

### Acceptance Criteria
- [ ] Pro upgrade screen accessible from Profile → "ترقية لـ Pro ⚡"
- [ ] Shows clear comparison: Free vs. Pro features
- [ ] Price displayed in EGP, equivalent to "أقل من حصة خصوصي واحدة في الشهر"
- [ ] Payment via: Credit/Debit card, Vodafone Cash, Fawry
- [ ] On successful payment → Pro badge shown on profile, limits lifted immediately
- [ ] Subscription auto-renews monthly; student can cancel anytime

**Priority:** P2

---

## US-13-03 · Pro Features Unlocked
**As a** Pro Student,
**I want to** access all Pro features without restrictions,
**so that** I get the full Neura experience.

### Acceptance Criteria
- [ ] Unlimited PDF pages/month
- [ ] Unlimited AI chat messages
- [ ] Neura voice notes (TTS)
- [ ] Detailed analytics (sleep correlation, subject performance)
- [ ] Concept maps
- [ ] Pro badge on leaderboard profile

**Priority:** P2

---

## US-13-04 · Subscription Management
**As a** Pro Student,
**I want to** manage my subscription,
**so that** I can cancel or change my plan if needed.

### Acceptance Criteria
- [ ] Settings → Subscription shows: plan type, renewal date, payment method
- [ ] "إلغاء الاشتراك" option with confirmation
- [ ] On cancellation → Pro access continues until end of billing period
- [ ] Cancellation confirmation sent via email/WhatsApp

**Priority:** P2
