# US-14 — Tutor Creators & B2B Features

---

## US-14-01 · Tutor Creator Account
**As a** Tutor,
**I want to** create a verified Neura Creator account,
**so that** I can upload my materials and reach my students on the platform.

### Acceptance Criteria
- [ ] "سجل كـ Creator" option on registration screen
- [ ] Verification process: submit national ID + teaching credentials
- [ ] Verified badge shown on tutor profile
- [ ] Creator account has a separate dashboard from student accounts

**Priority:** P3

---

## US-14-02 · Upload Tutor Materials
**As a** Tutor,
**I want to** upload flashcard sets and quizzes for my students,
**so that** they can practice my content on Neura.

### Acceptance Criteria
- [ ] Tutor can upload: Flashcard decks, Multiple-choice quizzes, PDF summaries
- [ ] Content is reviewed by Neura admin before going live (anti-spam)
- [ ] Tutor can set content as: Public (discoverable) or Private (invite-only via code)

**Priority:** P3

---

## US-14-03 · Affiliate Code for Students
**As a** Tutor,
**I want to** generate an affiliate code to share with my students,
**so that** they can access my private content on Neura.

### Acceptance Criteria
- [ ] Tutor generates a unique code from their dashboard
- [ ] Student enters code in Settings → "كود المدرس"
- [ ] Student gains access to that tutor's private content
- [ ] Tutor can see how many students used their code

**Priority:** P3

---

## US-14-04 · Sponsored Events Discovery
**As a** Student,
**I want to** discover relevant academic events (TEDx, NASA Space Apps, etc.) in the app,
**so that** I can participate in opportunities that match my interests.

### Acceptance Criteria
- [ ] Events section in the app (or card on dashboard)
- [ ] Events are targeted based on student's study subjects and location
- [ ] Event cards show: name, organizer, date, location, registration link
- [ ] Sponsored events are clearly labeled "برعاية [Sponsor]"
- [ ] Student can dismiss an event and not see it again
- [ ] No more than 1 sponsored card per session to avoid distraction

**Priority:** P3
