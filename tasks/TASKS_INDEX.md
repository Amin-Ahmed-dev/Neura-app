# Neura — Development Tasks Index

## Structure
Tasks are organized into **Epics** → **Sprints** → **Tasks**.
Each task has: ID, title, type, story reference, estimate, dependencies, and subtasks.

## Task Types
| Symbol | Type |
|--------|------|
| 🏗️ | Setup / Infrastructure |

| 🧠 | AI / ML Integration |
| 🗄️ | Database / Schema |
| 🔒 | Security / Auth |
| 🧪 | Testing |
| 📦 | DevOps / Config |

## Priority
| Label | Meaning |
|-------|---------|
| P1 | MVP — must ship in v1.0 |
| P2 | Important — ship in v1.1 |
| P3 | Nice to have — v2.0+ |
