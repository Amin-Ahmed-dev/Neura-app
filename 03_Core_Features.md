# Core Modules & Features

## 1. Adaptive Pomodoro & Focus Mode
- Dynamic Timer: Starts at 25/5, adapts based on user flow state.
- Focus Guard: Blocks social media apps during the timer (App Blocker/Accessibility API).
- Break Suggestions: Quick stretches, eye resting (20-20-20), or prayer synchronization based on local time.

## 2. Spaced Repetition & Active Recall
- AI Flashcards: Auto-generate Q&A cards from uploaded PDFs or images.
- Feynman Technique via Voice: User explains the answer via voice; AI analyzes the transcription and provides feedback on missing keywords.
- Concept Maps: Auto-generated visual trees for study materials (Semantic Chunking).

## 3. PDF & Material Processing
- Smart Chunking: AI splits PDFs based on semantic meaning (topics) not just page numbers.
- Math Step-by-Step: For math/physics, AI provides hints (first step of the rule) rather than solving it directly to prevent cheating.

## 4. Health & Sleep Tracking
- Face-down Tracking: Uses accelerometer to track sleep when phone is placed face-down.
- Smart Alarm: Wakes user at the end of a 90-min sleep cycle. Requires scanning a barcode (e.g., toothpaste) to dismiss the alarm.

## 5. Gamification (Neurons System)
- Currency: "Neurons" (earned via deep work and completing tasks).
- Leagues: Weekly leaderboards (School -> City -> National).
- Real-world Rewards: Redeem Neurons for mobile data packages, promo codes, or event tickets.

## 6. Time Management (AI Co-pilot)
- Guilt-free Rollover: Unfinished tasks are auto-redistributed across the week.
- Everest Method: Breaks down large goals (e.g., 95% in Math) into micro-steps.
